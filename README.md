# A simple project for practice
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok
  
API - Task manager with MySQL database

1. Create a task
2. Edit a task
3. List all tasks
4. List tasks by code
5. Delete a task

endpoints:
- POST create - api/todo/ 
- GET list all - api/todo/
- GET list by id - api/todo/{id}
- PUT update - api/todo/
- DELETE delete - api/todo/{id}
